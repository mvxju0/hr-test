# PROJECT: HR / 운영관리 시스템 목업

## 📌 프로젝트 목적
이 프로젝트는 실제 개발 전, 기능 시연 및 발표를 위한 HTML 기반 목업이다.

- 목표: 임원/관리자가 한눈에 이해할 수 있는 UI
- 실제 서비스처럼 보이는 수준의 완성도
- DB 연결 없이 더미데이터 기반
- 추후 실개발 전 구조 검증용

---

## 📁 파일 구조 (고정)

- index.html       : 로그인
- front.html       : 대시보드
- employee.html    : 직원관리
- org.html         : 조직관리
- onoff.html       : 온오프관리
- pos.html         : 매장
- office.html      : 사무실
- approval.html    : 가입/승인

---
